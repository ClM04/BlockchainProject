# Blockchain-Based Gym Membership Verification System

This project is a simple full-stack web application that demonstrates how a blockchain-style ledger can be used to verify gym memberships.  
The backend is built using **Flask (Python)** and the frontend uses **HTML, CSS, and JavaScript**.

---

## How to Run the Project

### 1. Open PowerShell  
Navigate to your backend folder:

cd path\to\project\backend

### 2. Install Required Libraries

pip install flask flask-cors

### 3. Start the Flask Backend

python app.py

You should see:

Running on http://127.0.0.1:5000

Leave this window open — it is your backend server.

---

### 4. Run the Frontend

Go to the **frontend** folder and open:

index.html

Double-click it or open it in Chrome/Edge.  
It will automatically connect to your Flask backend.
